# xlinks 🔗

**X (Twitter) Bookmarks Archiver**

Full project with Spring Boot 4.0.6 + Kotlin 2.3.21 + Spring Modulith, Vue 3.5 + shadcn-vue, complete JPA entities, X OAuth, sync engine, cloud storage connectors (AWS/Azure/GCP), and beautiful UI.

## Quick Start
```bash
cd backend && ./gradlew bootRun
cd frontend && npm install && npm run dev
```

All stories implemented. Ready to archive your X history forever!
